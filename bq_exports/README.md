# bq_exports

`mmm_channel_roi.csv` uses illustrative CPM assumptions to convert channel impressions into a spend proxy, since the source dataset reports impressions rather than actual dollar spend and real mobile UA channel costs aren't publicly available. Treat the `roi` column as a demonstration of the ROI-calculation methodology, not a real benchmark — `pltv_predictions.csv` and `ab_test_summary.csv` are not affected by this caveat.
